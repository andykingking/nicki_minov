# Help Nicki Minaj make songs!

## Use

* `ruby main.rb <band name>`
* e.g. `ruby main.rb Nicki Minaj`
* Experience the majesty.

## Niceties

* Downloads lyrics pretty quickly
* Caches lyrics
* Can be used with non-Nicki Minaj artists as well

## Improvements
